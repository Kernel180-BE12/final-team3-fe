## 💬 Q&A 준비

**Q: 실제 AI가 연동되어 있나요?**
A: 현재는 시뮬레이션이며, 백엔드 AI API 준비 시 즉시 연동 가능합니다.

**Q: 사용자 데이터는 어떻게 관리되나요?**  
A: JWT 토큰 기반으로 보안이 유지되며, 백엔드와 완전 연동되어 있습니다.

**Q: 모바일에서도 사용 가능한가요?**
A: 반응형 디자인이 적용되어 있어 모든 디바이스에서 사용 가능합니다.
